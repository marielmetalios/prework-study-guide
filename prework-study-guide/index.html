HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Content for the list item goes between the opening and closing li brackets.

CSS

  • CSS helps with page style.
  • CSS is a language used to style HTML elements.
  • CSS is responsible for the text style, size, positioning, color, and more on a website.
  • CSS is also what controls how a website's style shifts between desktop and mobile versions.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: tells us what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: bring us back to the main branch

JavaScript

  • Javascript creates dynamic content for websites.
  • It achieves this by adding new HTML elements while modifying existing ones.
  • Many coders enhance web development skills using JavaScript to create user-friendly and interactive websites.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a data structure that stores a collection of elements, all of the same data type, under a single variable name, allowing you to access individual elements within the collection using a unique index number, essentially acting like a numbered list where each item can be easily retrieved by its position in the sequence.
